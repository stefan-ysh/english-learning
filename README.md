## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: TailwindCSS, Magic UI (ShineBorder, BlurFade)
- **Animation**: Framer Motion, Canvas Confetti
- **State Management**: Zustand (with Persistence)
- **Audio**: Web Speech API (Optimized)

## Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/english-learning.git
   cd english-learning
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Open in browser**:
   Navigate to [http://localhost:3000](http://localhost:3000)
